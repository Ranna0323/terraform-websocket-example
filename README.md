## Terraform-websocket-example

2023.07

### Service Introduction
- multi-tenancy를 위한 자동화 배포(terraform)에 대한 테스트 코드
- AWS VPC를 생성하고 security group과 Internet gateway를 연결하는 간단한 terraform 스크립트 작성

### How to Use
- package.json 다운로드
  - npm install
- 프로그램 실행
  - node index.js

### Tech Stack

Backend : Node.js
